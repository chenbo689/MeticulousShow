# Meticulous 测试工具演示项目

这是一个使用 **Vite** 构建的复杂场景演示应用，用于展示 [Meticulous](https://meticulous.ai/) 测试工具的强大功能。

## 项目特点

- 🚀 使用 Vite 构建，开发体验极佳
- ⚡ React 18 + TypeScript
- 🎨 现代化的 UI 设计，响应式布局
- 📱 完整的复杂业务场景，适合自动化测试

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 [http://localhost:5173](http://localhost:5173) 打开。

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 使用 Meticulous 进行测试

### 安装 Meticulous

1. 访问 [Meticulous](https://meticulous.ai/) 并注册账户
2. 按照 Meticulous 的文档安装 SDK
3. 在项目中配置 Meticulous

### 测试场景示例

#### 电商系统测试流程
1. 浏览商品列表
2. 使用搜索功能查找特定商品
3. 按价格排序商品
4. 将商品添加到购物车
5. 调整商品数量
6. 进入结算页面
7. 填写订单信息
8. 选择支付方式
9. 提交订单
10. 验证订单成功提示

#### 数据看板测试流程
1. 查看默认数据展示
2. 切换时间范围（7天/30天/90天）
3. 刷新数据
4. 切换图表类型（销售额/访问量）
5. 导出数据
6. 查看最近活动

#### 表单系统测试流程
1. 填写个人信息（验证必填字段）
2. 填写公司信息
3. 选择兴趣领域
4. 上传文件
5. 进入确认页面
6. 验证所有信息
7. 提交表单
8. 查看成功提示

#### 用户管理测试流程
1. 搜索用户
2. 按角色筛选
3. 选择多个用户
4. 批量删除用户
5. 添加新用户
6. 编辑用户信息
7. 切换用户状态

## 技术栈

- **框架:** React 18
- **构建工具:** Vite 5
- **语言:** TypeScript
- **路由:** React Router v6
- **图标:** Lucide React
- **样式:** 原生 CSS

## 为什么选择这个项目展示 Meticulous？

1. **复杂的用户交互:** 包含多种交互模式（点击、输入、选择、拖拽等）
2. **丰富的状态管理:** 涵盖简单到复杂的状态管理场景
3. **异步操作:** 模拟真实应用中的数据加载和处理
4. **表单验证:** 展示各种验证场景和错误处理
5. **动态内容:** 展示动态渲染和条件渲染
6. **多步骤流程:** 完整的用户流程，适合端到端测试
